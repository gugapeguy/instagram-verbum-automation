## Notas de Uso

- Fontes primárias: consulta direta para validar fatos antes de publicar
- Fontes secundárias: sinal de novidades e análise de impacto
- Sempre citar a fonte primária na base de conhecimento — fontes secundárias são auxílio de descoberta, não de citação
- EXA MCP (Story 1.2) usará este arquivo para orientar as buscas automáticas
