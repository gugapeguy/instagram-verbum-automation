# Instagram Verbum Automation

Automação do Instagram para a Verbum Contabilidade — alinhada à API oficial da Meta.

## Estrutura

| Pasta | Conteúdo |
|-------|----------|
| `api/` | Scripts Node.js para Meta Graph API (publicação, métricas, agendamento) |
| `skills/` | Skills do Claude Code utilizadas no projeto |
| `roteiros/` | Scripts dos vídeos da Maria Clara para ElevenLabs + HeyGen |
| `templates/` | Estruturas de carrossel, copy e planejamento editorial |
