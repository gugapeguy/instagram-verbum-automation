## Story

**Como** CEO Master,  
**quero** que o CEO Criativo produza simultaneamente Design A (HTML preenchido) e Design B (prompt Canva completo) a partir do copy e brief do post,  
**para que** o pacote de aprovação entregue a Gustavo tenha sempre as duas opções de design prontas para escolha imediata.

## Acceptance Criteria

1. `skills/ceo-criativo.md` criada documentando o fluxo completo de geração de criativos
2. CEO Criativo recebe o brief completo do CEO Master: `[perfil] · [tipo] · [tema]` + copy humanizado do CEO Conteúdo (legenda + hashtags)
3. Design A: CEO Criativo seleciona o template correto de `templates/` com base no tipo de post (carrossel → `carrossel-educativo.html`, feed → `post-frase.html` ou `post-alerta.html`, reel → `reels-cover.html`, stories → `stories-cta.html`), preenche as variáveis `{{}}` com o copy recebido, e entrega HTML renderizável com path em `criatives/[tema-slug].html`
4. Design B: CEO Criativo gera prompt Canva completo (dimensões, paleta, tipografia, slide-a-slide com texto real) — layout ou destaque visual diferente de A (variação genuína)
5. Maria Clara: inserido `<!-- MARIA_CLARA_PLACEHOLDER -->` em templates que exigem avatar (reels, pilares de autoridade)
6. Checklist de brand guide da Story 3.4 é executado no HTML do Design A antes de entregar ao CEO Master
7. Output ao CEO Master:
   ```
   DESIGN A — HTML/CSS
   [thumbnail descritivo do slide principal] · criatives/[tema-slug].html
   Status: ✅ pronto / ⚠️ revisar

   DESIGN B — Prompt para Canva manual
   [prompt completo slide-a-slide]
   ```

## Dev Notes

**Mapeamento completo tipo → template:**
```
carrossel → templates/carrossel-educativo.html
reel      → templates/reels-cover.html
feed      → templates/post-frase.html (autoridade) OU templates/post-alerta.html (urgência)
stories   → templates/stories-cta.html
```

**Regra de diferenciação Design A vs B:**
- A: fundo navy #0D1B3E, título em gold, corpo em branco
- B: fundo navy #132860 (variação mais escura), título em creme #F2EDE3, corpo em gold claro #E8C97A

**Armazenamento de Design A:**
```
criatives/
└── [tema-slug]/
    └── design-a.html
```

Exemplo: `criatives/ibs-simples-nacional/design-a.html`

**Restrição:** CEO Criativo NUNCA inventa conteúdo. Usa exclusivamente o copy recebido do CEO Conteúdo para preencher as variáveis. O layout é de responsabilidade dele; o conteúdo é do CEO Conteúdo.

**Output ao CEO Master (formato exato, compatível com `docs/ceo-master-flow.md`):**
```
DESIGN A — HTML/CSS
[descrição do slide principal: fundo navy, título "[texto]" em gold...] · criatives/[slug]/design-a.html
Status: ✅ pronto

DESIGN B — Prompt para Canva manual
Dimensões: 1080x1080px | Total: N slides
[prompt completo]
```
