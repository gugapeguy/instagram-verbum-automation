## Story

**Como** CEO Master,  
**quero** uma skill de monitoramento legislativo (`skills/monitor-legislativo.md`) que usa EXA MCP para buscar atualizações nas fontes definidas em `knowledge/sources.md`,  
**para que** o pipeline de conteúdo sempre tenha informações recentes, categorizadas por pilar editorial e ranqueadas por relevância para o público @verbumcontabilidade.

## Acceptance Criteria

1. `skills/monitor-legislativo.md` criada documentando o comportamento, os prompts e as instruções de uso da skill
2. Skill aceita parâmetro de categoria (ex: `reforma-tributaria`, `simples-nacional`, `esocial`, `trabalhista`, `obrigacoes-acessorias`) e retorna lista estruturada de atualizações
3. Classificador de relevância atribui nível `alta / média / baixa` baseado em alinhamento com os 6 pilares editoriais do @verbumcontabilidade
4. Integração com fontes definidas em `knowledge/sources.md` (Story 1.1 obrigatória concluída)
5. Output estruturado para cada item: `título | resumo (max 150 palavras) | fonte URL | pilar | relevância | data`
6. Monitor retorna resultados de pelo menos 3 fontes diferentes por consulta
7. Skill documentada com exemplos de uso e integração com EXA MCP
8. Resultado do monitor é compatível com o formato de injeção no CEO Conteúdo (Story 2.2)

## Dev Notes

**Dependência obrigatória:**  
Story 1.1 DEVE estar com status Done antes de iniciar. `knowledge/sources.md` é referência essencial para as fontes a consultar.

**EXA MCP — como usar:**  
O EXA está disponível via Docker gateway:
```
Tool: mcp__docker-gateway__web_search_exa
Parâmetros: { query: string, numResults: number, type: "neural" | "keyword" }
```
Verificar se Docker gateway está rodando: `curl http://localhost:8080/health`  
Se indisponível: retornar mensagem de fallback ao CEO Master, NÃO travar o pipeline.

**6 Pilares Editoriais (brand guide):**
1. Reforma Tributária — IBS, CBS, IS, split payment
2. Educação Fiscal PF — IRPF, Carnê-Leão, declaração
3. Educação Fiscal PJ — IRPJ, CSLL, obrigações empresariais
4. Consciência Financeira — planejamento, reserva, fluxo de caixa
5. Autoridade/Humanização — bastidores, cases, Maria Clara em ação
6. Formatos Recorrentes — dicas rápidas, calendário, perguntas frequentes

**Lógica de classificação por pilar:**  
O classificador usa palavras-chave do título/resumo para mapear ao pilar. Exemplos:
- "IBS", "CBS", "Reforma Tributária" → Pilar 1
- "IRPF", "declaração", "carnê-leão" → Pilar 2
- "Simples Nacional", "IRPJ", "CNPJ" → Pilar 3
- "planejamento financeiro", "reserva" → Pilar 4
- Conteúdo sem encaixe claro → Pilar 6

**Restrição de conteúdo:**  
O monitor busca apenas notícias e atualizações. Não gera opinião. A interpretação do impacto é feita pelo CEO Conteúdo na Story 2.2.

**Graceful degradation:**  
Se EXA não retornar resultados suficientes (< 3 fontes), logar aviso e retornar o que tiver. NUNCA inventar resultados.
